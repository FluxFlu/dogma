Define

What is the define directive

The #define directive is used to create macros.





How to use

The define directive follows the syntax of #define [from] [to] as [name]. You can also use #define [from] [to], and the [name] of the macro will be the same as the [from]





What is a macro

A macro is a token that is substituted for another value at compile time.





Example: 

#define [x] [500]

console.log(x);

This will compile to:


console.log(500);




Looping during compilation

During compilation, the application of macros will loop until there are no changes. This means the following is possible: 


#define [x] [500]
#define [y] [x + 200]

console.log(y);

This will compile to:


console.log(500 + 200);




Arguments

Arguments can be used as part of macros, the same way one would do in a function. This is done with the syntax #define [$argument] [$argument] 


#define [add($x, $y)] [$x + $y]

console.log(add(1, 2));

This will compile to:


console.log(1 + 2);




Token Grouping

Tokens can be grouped so they are treated as a single argument for macro purposes. This is done with the syntax $(tokens) 


#define [add($x, $y)] [$x + $y]

console.log(add($(1 + 10), 2));

This will compile to:


console.log((1 + 10) + 2);

Macros cannot be used within token groups. This means the following does not compile: 

    
#define [x] [1]

#define [add($x, $y)] [$x + $y]

console.log(add($(x + 10), 2));
