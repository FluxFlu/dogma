With

What is the with directive

The #with directive is used to include a shebang in the file. If the `make-script` compiler flag is used on a file using #with, the default language shebang will not be added.





How to use

The with directive follows the syntax of #with [shebang]





Example: 

#with [/usr/bin/node] // This file will use `#!/usr/bin/node` instead of the default `#!/usr/bin/env node`
